HTML

HTML is the standard markup language used for creating web pages. It defines the structure and content of a webpage using elements and tags. Elements are enclosed in angle brackets and consist of opening and closing tags.

CSS

CSS is a styling language used to control the presentation and layout of web pages. It separates the content (HTML) from its visual appearance, allowing for design customization. CSS rules define properties like color, font, spacing, and positioning.

JAVASCRIPT

JavaScript is a versatile scripting language used for adding interactivity to web pages. It can manipulate HTML and CSS, handle user interactions, and perform dynamic actions. JavaScript is executed in the browser, enabling client-side scripting.

JQUERY

jQuery is a JavaScript library designed to simplify DOM manipulation and event handling. It provides a concise and cross-browser-compatible way to interact with web page elements. jQuery simplifies common tasks like animation, AJAX requests, and event binding.

BOOTSTRAP

Bootstrap is a popular front-end framework for building responsive and visually appealing websites. It includes a collection of CSS and JavaScript components, such as grids, forms, and modals. Bootstrap streamlines web development and ensures consistent design across devices.
